# Homepage Project

A clean, responsive website built with HTML, CSS, and JavaScript.

## Files

- **index.html** - The website
- **styles.css** - Styling and layout
- **script.js** - Interactive features
- **README.md** - How to use this project
- **GETTING_STARTED.md** - Quick start guide
- **AUDIT.md** - Project requirements checklist
- **TESTING.md** - How the project was tested
- **GITHUB_PAGES.md** - Deployment information

## Quick Start

Open `index.html` in your browser.

For a local server:
```bash
python3 -m http.server 8000
```

Visit http://localhost:8000
